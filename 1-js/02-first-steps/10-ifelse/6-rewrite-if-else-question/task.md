# Réécrire 'if..else' en '?'

Réécrire ce `if..else` en utilisant plusieurs opérateurs ternaires `'?'`.

Pour plus de lisibilité, il est recommandé de diviser le code en plusieurs lignes.

```js
let message;

if (login == 'Employé') {
  message = 'Bonjour';
} else if (login == 'Directeur') {
  message = 'Salutation';
} else if (login == '') {
  message = 'Pas de connexion';
} else {
  message = '';
}
```
